РЕКОМЕНДАЦИИ ПО ВИДЕО:
================================================================================

hero-background.mp4:
- Разрешение: 1920x1080 (Full HD) или 1280x720 (HD)
- Формат: MP4 (H.264 кодек)
- Длительность: 15-30 секунд (будет зациклено)
- Размер файла: до 10 МБ для быстрой загрузки
- Содержание: студенты за компьютерами, кампус, лаборатории, технологии
- Без звука (видео автоматически приглушено)

Если видео не добавлено, будет показан синий градиентный фон с placeholder.


РЕКОМЕНДАЦИИ ПО ИЗОБРАЖЕНИЯМ:
================================================================================

1. Формат: JPG для фотографий, PNG для логотипов и графики
2. Оптимизация: сжимайте изображения для веба (TinyPNG, ImageOptim)
3. Соотношение сторон:
   - Широкие (gallery wide): 2:1
   - Обычные: 3:2
   - Высокие (gallery tall): 3:4

4. Качество: достаточно 72-150 DPI для веба


АВТОМАТИЧЕСКАЯ ОБРАБОТКА ОТСУТСТВУЮЩИХ ИЗОБРАЖЕНИЙ:
================================================================================

Если изображение не найдено, на его месте автоматически появится:
- Для фото: placeholder с иконкой камеры и путём к файлу
- Для видео: градиентный фон с подсказкой о добавлении видео
- Для логотипа: только текст "САФУ"
